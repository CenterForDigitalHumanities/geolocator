Welcome to the RERUM Geolocator

Here you can enhance Linked Open Data resource by providing geospatial descriptors. This does not affect the original resource nor does it notify the resource owner of the enhancement. There is no user system and so data generated here cannot be attributed to you specifically. However, it will be available to you and other applications to make it your own.

IIIF Resource

This application is proud to supoort the IIIF Presentation API and encourages you to bring your own IIIF resources. If your web resource fits this category, there are two main paths to take.

Web Annotation

The IIIF Presentation API incorporates the W3C Web Annotation for the "Annotation" objects created under its schema. This Annotations can be embedded directly onto IIIF Defined Types. It can also target those data types as non-destructive, non-dispruptive independent data assertion. This process seeks to finish with a real active LOUD Geolocating Web Annotation for the resource you provide.

navPlace

The IIIF Presentation API also incorproates community extensions. The IIIF Maps groups developed a foundation around GeoJSON-LD as a data format to support within the IIIF APIs and data schemas. The navPlace Extension formally introduced a `navPlace` property which is designed to house a single GeoJSON Feature Collection thus making a formal and direct connection between the resource and place.

Other Linked Open Data Resources

Linked Open Usable Data is a vast expanse of different data classifications and well described taxonomies. If your URI resolves to JSON, preferably JSON-LD (what's the difference?), then you can provide GeoJSON-LD for that LOUD entity through a non disruptive process using a W3C Web Annotation. This process seeks to finish with a real active LOUD Geolocating Web Annotation for the resource you provide.
